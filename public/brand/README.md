# Personal Branding Photos

Drop personal-branding photos here as **.webp** files using these exact names so
they appear automatically across the site (About profile card photo strip):

- `portrait-1.webp` — studio / headshot
- `portrait-2.webp` — at the workstation
- `portrait-3.webp` — candid / outdoor
- `portrait-4.webp` — presenting / speaking
- `portrait-5.webp` — casual

Recommended: square or 3:4 portrait crops, ~1000px on the long edge, compressed
WebP (quality ~80). Until real photos are added, the UI shows tasteful gradient
placeholders with the "FR" monogram — nothing breaks.
